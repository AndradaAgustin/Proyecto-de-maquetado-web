# 📘 Proyecto de Maquetado Web

## 📌 Descripción
Este proyecto es un **trabajo práctico** en el cual se crean métricas a las que luego le agregaremos **JavaScript** para añadir interactividad y dinamismo.

## 🛠️ Tecnologías utilizadas
- **HTML5** → estructura del contenido de la aplicación.  
- **CSS3** → diseño visual, estilos y distribución de los elementos en la interfaz.  

## 📂 Recursos
- Imágenes e íconos almacenados localmente en la carpeta `assets/images`.  
- Diseño basado en **componentes visuales simples**, utilizando contenedores y tarjetas informativas.  

## 🚀 Próximos pasos
- Integración de **JavaScript** para añadir funcionalidades dinámicas.  
- Optimización del diseño para mejorar la experiencia del usuario.
